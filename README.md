#### Debug keys

| Scene | Key | Function |
| - | - | - |
| Main | H | Successful hit |
| Main | Y | Two successful hits in quick succession |
| Main | M | Missed hit |
| Main | P | Start game |
| Main | X | Zoom out |
| Main | Z | Zoom in |
| Score | W | Increase counter |
| Score | S | Decrease counter |
